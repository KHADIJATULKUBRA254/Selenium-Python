Daraz E-commerce Automation Testing (Selenium + Python)

Project Overview
This project automates key test scenarios of the Daraz e-commerce website using Selenium WebDriver with Python and PyTest.
The automation focuses on validating laptop search results, brand filtering, price filtering, product count, and free shipping availability.

The project follows the Page Object Model (POM) to ensure clean, maintainable, and reusable test code.
Tools & Technologies
-Python
-Selenium WebDriver
-PyTest
-Page Object Model (POM)
-Automated Test Scenarios

1. Laptop Search & Filter Validation
-Search for "laptop"
-Apply brand filter
-Apply price range filter
-Verify products are displayed
-Validate product count is greater than zero

2. Free Shipping Verification
-Search for "laptop"
-Apply brand & price filters
-Open first product from results
-Verify free shipping availability

Key Learning Outcomes
-Hands-on experience with Selenium automation
-Implementation of Page Object Model
-Automated testing of real-world e-commerce features
-Improved understanding of QA workflows and validation logic
